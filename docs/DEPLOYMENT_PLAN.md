# LLM 交易策略分析器 - 部署方案

## 一、项目背景

### 现有项目
- 基于 Python 的 LLM 量化交易策略分析系统
- 使用 Gradio UI + FastAPI 后端
- 数据来源：AKShare（A股/港股/美股/期货）
- LLM 决策：支持 Claude、OpenAI 等

### 目标
- 部署为 Web 服务
- **低价多销** - 目标小红书散户（投资4-5万，预算敏感）
- **完全匿名** - 不备案，海外运营
- **保护算法** - 所有逻辑在后端，不暴露

### 套餐设计

| 套餐 | 定价 | 次数限制 | API成本 | 利润 | 目标 |
|------|------|----------|---------|------|------|
| **免费版** | ¥0 | 每天 1 次 | ¥0.6/月 | -¥0.6 | 引流体验 |

| **高级版** | ¥19/月 | 每天 15 次 | ¥9/月 | **¥10** | 重度用户 |

- **¥9/月超低价** - 小红书散户也愿意付

### 收入预估

| 用户数 | 基础版收入 | API成本 | 利润 |

| 100 用户 | ¥900/月 | ¥300/月 | ¥550/月 |
| 500 用户 | ¥4500/月 | ¥1500/月 | ¥2950/月 |

**只要有 10 个付费用户，基本覆盖服务器成本。**
