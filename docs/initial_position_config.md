# 初始持仓配置功能说明

## 功能概述

现在支持在回测和最新K线分析模式中配置初始持仓，让AI决策时可以考虑当前持仓情况。这对于以下场景非常有用：

1. **回测中途接入**：模拟从某个已有持仓的状态开始回测
2. **实时决策辅助**：输入当前实际持仓，让AI给出下一步操作建议
3. **持仓管理策略测试**：测试不同持仓状态下的AI决策表现

## 命令行参数

### `--initial-position` 
初始持仓数量
- **类型**: 整数
- **默认值**: 0（空仓）
- **说明**: 
  - 正数表示多头持仓（如 `2` 表示持有2手多单）
  - 负数表示空头持仓（如 `-3` 表示持有3手空单）
  - 0 表示空仓

### `--entry-price`
初始持仓的开仓价格
- **类型**: 浮点数
- **默认值**: 0.0
- **说明**: 
  - 当设置了 `--initial-position` 时，必须同时设置有效的开仓价格
  - 用于计算浮动盈亏和止损止盈点位

## 使用示例

### 1. 回测模式 - 多头持仓

假设你在5000点做多了2手，想看AI后续如何处理：

```bash
python src\backtest\llm_decision_backtest.py --start 2024-09-01 --end 2024-10-31 --initial-position 2 --entry-price 5000 --show_rationale
```

### 2. 回测模式 - 空头持仓

假设你在5200点做空了3手：

```bash
python src\backtest\llm_decision_backtest.py --start 2024-09-01 --end 2024-10-31 --initial-position -3 --entry-price 5200 --show_rationale
```

### 3. 最新K线分析 - 咨询持仓建议

查看当前持有2手多单（开仓价5100）时，AI的最新建议：

```bash
python src\backtest\llm_decision_backtest.py --latest --initial-position 2 --entry-price 5100 --show_rationale
```

### 4. 最新K线分析 - 空头持仓建议

查看当前持有1手空单（开仓价5300）时的建议：

```bash
python src\backtest\llm_decision_backtest.py --latest --initial-position -1 --entry-price 5300 --decision-period 1440 --auxiliary-periods 60,240
```

## AI决策时会考虑的信息

当配置了初始持仓后，AI在做决策时会获得以下信息：

1. **持仓基本信息**
   - 持仓方向（多头/空头）
   - 持仓数量
   - 开仓价格
   - 当前价格

2. **盈亏情况**
   - 浮动盈亏金额
   - 浮动盈亏百分比
   - 价格变动幅度

3. **风险控制信息**
   - 当前止损点位（如果设置）
   - 当前止盈点位（如果设置）
   - 距离止损/止盈的距离

4. **账户状态**
   - 可用资金
   - 保证金占用
   - 风险度

## 注意事项

1. **参数验证**：如果设置了 `--initial-position` 但没有设置 `--entry-price`（或设置为0），程序会报错退出

2. **持仓方向**：
   - 正数 = 多头（看涨）
   - 负数 = 空头（看跌）
   - 零 = 空仓

3. **价格单位**：开仓价格应使用与合约一致的单位（如郑棉SA合约单位是元/吨）

4. **回测起点**：初始持仓会在回测开始时立即建立，AI会从第一根K线开始考虑这个持仓

5. **多周期分析**：初始持仓功能与多周期分析完全兼容，可以同时使用

## 常见问题

**Q: 为什么需要同时设置数量和价格？**  
A: 价格用于计算浮动盈亏和风险点位，这对AI做出合理决策非常重要。

**Q: 初始持仓会影响初始资金吗？**  
A: 不会。初始资金仍然按 `--initial_units` 或 `initial_capital` 计算，初始持仓只是告诉AI你当前的持仓状态。

**Q: 可以中途改变持仓信息吗？**  
A: 不可以。初始持仓在回测开始时设置一次，之后由AI决策和市场变化来改变持仓。

**Q: 支持同时持有多空吗？**  
A: 不支持。期货交易通常是单向持仓（多或空），不能同时持有多空。
