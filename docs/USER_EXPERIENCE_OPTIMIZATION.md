# 用户交互体验优化方案 v2.0

## 核心目标

| 目标 | 策略 |
|------|------|
| **简单上手** | 3秒上手，输入股票→点分析→看结论 |
| **服务专业** | 显示专业术语和指标，显得专业 |
| **超值感受** | ¥9/月，每天5次，比雇分析师便宜100倍 |
| **提高付费** | 引导自然，不强迫 |

## 一、金融小白心理分析

### 他们的痛点与解决方案

| 痛点 | 解决方案 |
|------|----------|
| 不知道买什么 | → 热门股票推荐+AI智能选股 |
| 怕买错亏钱 | → 给出止损价，明确风险 |
| 看不懂K线 | → 给结论+专业术语（显得专业） |
| 不知道准不准 | → 显示置信度 |
| 怕被割韭菜 | → ¥9/月白菜价，试错成本低 |

### 他们的期待

1. **像点外卖一样简单** - 输入股票 → 得出结论
2. **像看天气一样直观** - 买入/观望/卖出 + 置信度
3. **像有顾问一样放心** - 有专业分析支撑（术语显得专业）

## 五、临时持仓信息（不存储）

### 设计原则

**用户隐私优先：**
- ❌ 不注册账号
- ❌ 不存储持仓数据
- ❌ 不需要登录
- ✅ 每次分析时临时输入
- ✅ 关闭页面后数据消失

### 信息输入方式

#### 方式1: 逐项填写

```
持有数量: [100] 股
成本价:   [1700] 元
本股计划投入: [200000] 元
最大持仓: [200] 股
```

#### 方式2: 快速输入（自然语言）

```
用户输入: "我持有100股，成本1700，计划投20万，最多买200股"

系统解析:
✓ 持有数量: 100股
✓ 成本价: 1700元
✓ 本股计划投入: 200000元
✓ 最大持仓: 200股
```

### AI 根据信息计算

```
输入信息:
- 持有: 100股 @ ¥1700 = ¥170,000
- 计划投入: ¥200,000
- 最大持仓: 200股
- 当前股价: ¥1,780

AI 计算:
- 已投入: ¥170,000 (85%)
- 剩余可用: ¥30,000 (15%)
- 可再买: 16股
- 建议操作: 不建议加仓，仓位已偏高

- 止损价: ¥1,650 (下跌7.3%)
- 止盈价: ¥1,950 (上涨9.6%)
- 风险/收益比: 1:1.3
```

### 不存储的好处

| 对比项 | 存储持仓 | 不存储（推荐） |
|--------|----------|----------------|
| 隐私安全 | ❌ 泄露投资信息 | ✅ 完全隐私 |
| 复杂度 | 需要注册/登录 | ✅ 零门槛 |
| 数据维护 | 需要更新持仓 | ✅ 无需维护 |
| 用户体验 | 长期跟踪 | 每次临时输入 |
```

def smart_search(keyword: str) -> List[dict]:
    """智能股票搜索，支持多种输入格式"""
    keyword = keyword.strip().upper()

    # 1. 纯数字 → 股票代码 (6位A股 / 5位港股)
    if keyword.isdigit():
        if len(keyword) == 6:
            return search_a_stock(keyword)
        elif len(keyword) == 5:
            return search_hk_stock(keyword)

    # 2. 纯英文 → 美股
    if keyword.isalpha() and keyword.isascii():
        return search_us_stock(keyword)

    # 3. 中文 → 名称搜索
    if is_chinese(keyword):
        return search_by_name(keyword)

    # 4. 无结果 → 返回热门股票
    return get_hot_stocks()

### 4. 仓位计算逻辑

```python
def calculate_position(user_balance: float, stock_price: float,
                       stop_loss_pct: float, preset: str) -> dict:
    """根据用户资金和风险偏好计算建议仓位"""

    # 根据预设确定仓位比例
    position_pct = {
        "conservative": 0.20,  # 保守型 20%
        "balanced": 0.30,      # 平衡型 30%
        "aggressive": 0.50,     # 进取型 50%
    }.get(preset, 0.30)

    # 计算可买入数量
    max_amount = user_balance * position_pct
    quantity = int(max_amount // stock_price)  # 取整手

    # 计算止损金额
    stop_loss_price = stock_price * (1 - stop_loss_pct)
    max_loss = (stock_price - stop_loss_price) * quantity

    # 计算止盈 (目标涨幅)
    target_pct = 0.05 if preset == "conservative" else 0.08 else 0.12
    target_price = stock_price * (1 + target_pct)
    expected_gain = (target_price - stock_price) * quantity

    return {
        "quantity": quantity,
        "amount": quantity * stock_price,
        "position_pct": position_pct,
        "stop_loss_price": stop_loss_price,
        "stop_loss_amount": max_loss,
        "target_price": target_price,
        "expected_gain": expected_gain,
        "risk_reward_ratio": abs(expected_gain / max_loss) if max_loss > 0 else 0
    }
```
